# QR Dine App

![React](https://img.shields.io/badge/React-19-61DAFB?logo=react)
![Vite](https://img.shields.io/badge/Vite-5-646CFF?logo=vite)
![Docker](https://img.shields.io/badge/Docker-Ready-2496ED?logo=docker)
![License](https://img.shields.io/badge/License-Private-red)

**QR Dine** is a modern multi-tenant SaaS platform that enables restaurants, cafés, and retail stores to create and manage QR-based digital menus.

This repository contains the **frontend application (QR Dine UI)** built with React and Vite. It provides the user interfaces for store management, digital menu presentation, and administrative operations.

Through this UI, businesses can:

- 📱 Deliver digital QR menus to customers
- 🛠 Manage menu items in real-time
- 🏪 Operate within their dedicated store environment
- 📊 Access management and monitoring tools

The frontend communicates with the QR Dine backend via RESTful APIs and is designed with a scalable, feature-based architecture to support long-term growth and maintainability.
